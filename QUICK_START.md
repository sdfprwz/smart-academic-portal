# 📖 Quick Start Guide - Smart Academic Portal

## Getting Started (Easy!)

### Step 1: Open the Website
Simply open `index.html` in any web browser (Chrome, Firefox, Safari, Edge)

### Step 2: Explore the Homepage
You'll see:
- **Smart Academic Portal** header
- **Quick Access Theme & Language Buttons** (top right)
- **Login option** via the side menu (🔒) or simply click a role card – you'll be asked to authenticate
- **Role Selection Cards** (center) - Choose your role (after logging in if required)
- **Key Features Section** (below)
- **Quick Links Section** (at bottom)

> Note: A dedicated `login.html` page handles user role selection and stores your session; you are automatically redirected when you select a role card or if access is required for a page.

## 🎯 Quick Tour

### Path 1: I'm a Student
1. Click the **"Student"** card on homepage
2. Explore the student dashboard:
   - 📢 Recent Notices
   - 📚 Syllabus (4 semesters)
   - ⏰ Weekly Timetable
   - 📥 Forms & Downloads
   - 📊 Results & Grades
   - ✅ Attendance Info

### Path 2: I'm Faculty
1. Click the **"Faculty"** card
2. Navigate the faculty portal:
   - 👤 View faculty profile
   - 📚 Manage your courses
   - 📤 Upload course materials
   - 📢 Publish course notices
   - ✅ Mark attendance
   - 🔬 Research profile

### Path 3: I'm a Research Scholar
1. Click the **"Research Scholar"** card (or use the login link in the menu)
2. You’ll be taken to the login page. After authenticating you land on the research dashboard:
   - 📋 View your research projects
   - 👨‍🏫 Info about your supervisor
   - 📄 Publications & uploads
   - 🎤 Conferences & seminars
   - 💰 Funding & scholarships
   - 📊 PhD progress tracking

> General information about research (programs, projects, how to join) is publicly available via the **Research (Public)** tab in the side menu. No login required for that page.

### Path 4: I'm an Administrator
1. Click the **"Administration"** card
2. Access admin panel:
   - 📊 System overview & statistics
   - 📢 Create/manage notices
   - 👥 User management
   - 🏢 Department management
   - 🗄️ Backup & archive
   - 📈 Reports & analytics

### Path 5: I'm a Visitor
1. Click the **"Visitor"** card
2. Explore department information:
   - 📘 About the department
   - 👨‍🏫 Faculty directory
   - 📚 Academic programs
   - 📅 Department timetable
   - 🏆 Achievements
   - 📞 Contact information

## ⭐ Key Features to Try

### 1. **Dark Mode** 🌙
- Click the "🌙 Dark Mode" button (top right)
- Watch the entire site transform to dark theme
- Click again to return to light mode
- Your preference is saved automatically!

### 2. **Language Switching** 🌍
- Click the "हिंदी / English" button (top right)
- All text changes to Hindi
- Click again to go back to English
- Works on all pages!

### 3. **Smart Search** 🔍
- Try these search terms:
  - `"notice"` → Goes to Notice Board
  - `"exam"` → Goes to Notices
  - `"faculty"` → Goes to Faculty Portal
  - `"department"` → Goes to Department Page
  - `"calendar"` → Goes to Academic Calendar
  - `"syllabus"` → Goes to Student Page

### 4. **Notice Filtering** 📌
- Go to **Notices** page (from homepage or student page)
- Click different filter buttons:
  - "All Notices" - See everything
  - "🔴 Urgent" - Red priority items
  - "🔵 Academic" - Course-related items
  - "🟡 Events" - Event announcements

### 5. **Responsive Mobile Design** 📱
- Open on a mobile phone OR
- Resize your browser window to smaller widths:
  - Desktop: Full layout
  - Tablet (< 768px): Adjusted layout
  - Mobile (< 480px): Single column layout

### 6. **Keyboard Navigation** ⌨️
- Press **Tab** to navigate through all interactive elements
- Press **Enter** to click buttons/links
- Notice the focus indicators around elements
- Try accessing with keyboard only (no mouse)

## 🎨 Design Highlights

### Color Coding System
- 🔴 **Red/Urgent**: Important deadlines, urgent notices
- 🔵 **Blue/Academic**: Course-related, academic info
- 🟡 **Yellow/Events**: Events, workshops, seminars
- 🟢 **Green/Success**: Positive status, completed items
- 🟠 **Orange/Warning**: Warnings, pending items

### Visual Elements
- **Cards**: Hover to see effects (elevation, color change)

- **Forms**: Clean input fields with focus effects
- **Buttons**: Clear call-to-action styling
- **Badges**: Status indicators with color coding

## 💡 Smart Features Explained

### 1. Role-Based Navigation
Instead of showing everything, each user sees only what's relevant:
- Students see: Notices, grades, attendance
- Faculty see: Course management, student info
- Researchers see: Projects, publications, grants
- Admin see: System management tools

### 2. Notice Board Intelligence
- **Categorized**: Urgent vs Academic vs Events
- **Priority Display**: Urgent notices appear first
- **Auto-expiry**: Shows expiration dates
- **Filtering**: Quickly find what you need

### 3. Multilingual Interface
- Seamless switching between English & Hindi
- All text translates dynamically
- Settings save automatically
- Works across all pages

### 4. Dark Mode
- Reduces eye strain in low-light conditions
- Professional appearance
- Increases battery life on OLED devices
- Preference persists across sessions

### 5. Mobile Responsive
- Single breakpoint adjustments
- Touch-friendly buttons
- Readable on all screen sizes
- Fast loading

### 6. Accessibility First
- Keyboard-navigable
- Screen reader compatible
- High contrast options
- Semantic HTML structure

## ❓ FAQ

**Q: Can I change user roles?**
A: Yes! Click the back button or navigate to index.html to switch roles.

**Q: Is this a real working website?**
A: It's a prototype with realistic mock data. A real version would add database backend.

**Q: Do the buttons actually work?**
A: Navigation buttons work. Action buttons (like "Upload") show confirmation messages.

**Q: Can I print the pages?**
A: Yes! The CSS is print-friendly. Use Ctrl+P to save as PDF.

**Q: Works offline?**
A: Yes! Everything is local HTML/CSS/JS. No internet needed.

**Q: Can I modify it?**
A: Absolutely! It's open source prototype. Edit HTML/CSS/JS freely.

**Q: How do I add new content?**
A: Edit the HTML files directly. Add new cards, sections, or notices easily.
